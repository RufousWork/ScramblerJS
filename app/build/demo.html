How to use the scrambler

  1. select a DOM element: b = document.querySelector('body')
  2. initialize and start the scrambler: scrambler.scramble(b)
  3. stop the scrambler: scrambler.stop()
  4. restore the original text: scrambler.restore()

Sample text

The quick brown fox jumped over the lazy dog.
