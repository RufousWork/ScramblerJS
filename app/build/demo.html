For the impatients

  1. add the scrambler.js and jquery libraries to the page you want to scramble
  2. then run the presets passing a locale for the CTA: scrambler.go('en')

this operation will scramble all text nodes under the main body element and show the CTA after a preset interval.

How to use the scrambler

the scrambler can run manually by calling each individual function exposed by the library:

  1. add the scrambler.js and jquery libraries to the page you want to scramble
  2. select the root element that needs to be scrambled, e.g.: el = document.querySelector('.article')
  3. initialize and start the scrambler: scrambler.scramble(el). Note! the CTA at the end of the scrambling (read below) can be shown automatically by passing a boolean parameter, e.g. scrambler.scramble(el, true)
  4. stop the scrambler: scrambler.stop()
  5. show the CTA passing a locale: scrambler.showCTA('en')
  6. restore the original text: scrambler.restore()

NOTE 1. The scrambler can work without jquery. Jquery is only used to show the CTA with defined timings and localized copy.

Note 2. All the copy elements are defined in file copy.json.
